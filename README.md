# Website Hỗ trợ Học tiếng Anh THPT

## 📚 Giới thiệu
Website hỗ trợ học sinh THPT học tiếng Anh với:
- 🤖 Chatbot AI sửa lỗi viết và định hướng học tập
- 📖 Tài liệu ôn thi giữa kỳ, cuối kỳ lớp 10, 11, 12
- 📝 30+ bài tập trắc nghiệm, đọc hiểu, viết lại câu theo từng unit

## 🚀 Công nghệ sử dụng
- Frontend: HTML, CSS, JavaScript (GitHub Pages)
- Backend: Vercel Serverless Functions
- AI: Google Gemini API

## 📁 Cấu trúc dự án
- `/frontend` - Mã nguồn frontend
- `/api` - API endpoints (Vercel)
- `/data` - Dữ liệu bài tập và tài liệu
